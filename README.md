# 🚀 ETL Pipeline Profissional (Nível Contratação)

## 📌 Descrição
Pipeline ETL completo com boas práticas de engenharia de dados.

## 🔄 Etapas
- Extract: leitura de CSV
- Transform: limpeza, validação e enriquecimento
- Load: persistência em SQLite

## 🛠 Tecnologias
- Python
- Pandas
- SQLite
- Logging

## ▶️ Execução
```bash
pip install -r requirements.txt
python main.py
```

## 💼 Diferenciais
- Separação de camadas (raw / processed)
- Logging estruturado
- Tratamento de erros
- Validação de dados
- Configuração centralizada

## 📂 Estrutura
```
data/
 ├── raw/
 ├── processed/
etl/
db/
```
